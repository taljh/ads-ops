## 🗺️ خريطة الطريق (المراحل) — مرتّبة بالأولوية

### المرحلة 0 — التأسيس (هذا الأسبوع) ✅ جارٍ
- [x] إضافة العميلة بفصل تام + سحب التاريخ الكامل (TikTok + Snapchat).
- [x] تشخيص عميق 5 طبقات + خط أساس + winners/losers.
- [ ] **مدخلات المستخدم:** سلة + سقف الميزانية + الهامش + المنصات الأخرى (تُغلق التارقت والأرقام أدناه).

### المرحلة 1 — وقف النزيف وتثبيت المحرّك (الأسبوع 1–2) 🔝
1. 🔴 **معالجة إجهاد Snapchat (الأعجل):** 2–3 كرييتف جديد لمجموعة Wasayef + توسيع الجمهور (Lookalike/اهتمامات أوسع) + سقف تردد → كسر الـ10.3 نحو <6.
2. 🟢 **TikTok:** استمرار طبقة Lookalike الناجحة + تغذية Wasayef بنسخ من الكرييتف البطل (Image carousel 3) قبل ما يتعب.
3. 🟢 **استمرارية:** لا انقطاعات صرف (درس 10–15 مايو) + متابعة pacing يومية.
4. 🔴 **تقليم الذيل:** إعلانات "#اكسبلور/#fyp" بـ0–1 شراء + استثناء الصفر (≥2× CPA و0 تحويل = أوقف).
- **هدف نهاية المرحلة:** Snapchat تردد <6 مع الحفاظ على ROAS؛ TikTok ROAS ≥5 بثبات.

### المرحلة 2 — سدّ الفجوات الكبرى (الأسبوع 3–4) 🔝
1. 🔝 **بناء Retargeting** (غائب كلياً): جمهور زوار + سلات متروكة + مشاهدي فيديو (آخر 14–30 يوم)، منفصل عن Cold، يبدأ ~10–15% ميزانية. (درس زين: عادةً أعلى ROAS.) لأي منتج؟ يُقاد بـ **Wasayef** أولاً (أكبر بركة نية شرائية).
2. 🔝 **حملة موسم رواتب** مخصّصة للذروة 26→3 (بنية روابح مثبتة).
3. 🟢 **توسيع الكرييتف البطل** (Image carousel 3) بتنويعات (أول 3 ثوانٍ/خلفية/CTA).

### المرحلة 3 — تنويع وتقليل الهشاشة (الشهر 2) 
1. 🔝 **تقليل تركّز Wasayef** (52–94%): قوِّ Asayel كمحرّك ثانٍ + اختبر/أحيِ موديلات إضافية بكرييتف جديد.
2. 🟢 **cross-sell / رفع AOV** + تحسين صفحة المنتج (PDP) بعد توفّر بيانات سلة.

### المرحلة 4 — توسيع متدرّج بالكفاءة (الشهر 2–3+)
- **قاعدة التوسيع (إلزامية بعد درس سبتمبر):** TikTok **+20–30% كل 2–3 أيام** على الرابح فقط · لا قفزات. Snapchat بحذر أكبر (حساس للتردد).
- **قرار scale/hold/cut:** Blended فوق التارقت → وسّع · تحته → أصلح الكرييتف/اللاندنق قبل أي زيادة.

| الشهر | صرف مستهدف | ROAS منصة (حد أدنى) | Blended (هدف) | منطق |
|---|---|---|---|---|
| مايو (فعلي) | ~9.0K | TT 5.55 / Snap 8.18 | ⏳ | خط الأساس |
| يونيو | ⏳ (بعد السقف) | ≥5.0 / ≥6.0 | ≥6.0 | تثبيت + وقف إجهاد Snapchat |

**قاعدة الرامب:** أي شهر ينزل فيه ROAS تحت الأرضية → نوقف الزيادة ونصلح. **لا نكرر سبتمبر 2025.**
